DESCRIPCIÓN DEL PROYECTO:

El siguiente proyecto realiza lo siguiente:
- Importa los datos de los archivos csv a una base de datos MySQL.
- En Mage AI, exporta los datos a una base de datos en Snowflake
- Los datos crudos de SnowFlake pasan por una tubería ELT y se pasan en limpio a un esquema diferente.
- La limpieza fue gracias al jupyter notebook "eda.py" que toma los datos en crudo de Snowflake y presenta diferentes características de estas.
- El archivo "insights" muestra las preguntas del literal 6 del PDF.
